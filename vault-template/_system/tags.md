# Tag Taxonomy

*The allowed tag whitelist. The `tag-taxonomy` and `wiki-lint` skills enforce this — new tags must be added here before use, to prevent tag sprawl.*

## Categories (frontmatter `category:`)

- `concept` — ideas, theories, mental models
- `entity` — people, organizations, tools, products
- `reference` — summaries of a specific source
- `insight` — your own synthesized takeaways
- `synthesis` — cross-cutting analysis spanning multiple sources

## Topic Tags (frontmatter `tags:`)

*Add your domain tags here as the wiki grows. Examples:*

- `llm`
- `knowledge-management`
- `obsidian`
